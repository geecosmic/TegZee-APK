## Overview
TegZee is a modern ride‑hailing platform connecting passengers with verified drivers.

## Core Features
### Passenger
- Request rides
- Track drivers
- Wallet funding
- Schedule rides
- Saved places
- Ratings & SOS

### Driver
- Verification
- Earnings dashboard
- Withdrawals
- Vehicle management

### Admin
- User management
- Ride monitoring
- Platform wallet

## Stack
| Layer | Technology |
|---|---|
| Frontend | Flutter |
| Backend | Django |
| Database | Supabase |
| Payments | Paystack |
| Auth | Supabase Auth |

## Setup

### Flutter
```bash
flutter pub get
flutter run
```

### Django
```bash
python manage.py migrate
python manage.py runserver
```
